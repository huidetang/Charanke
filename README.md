Gribouillageは、メモを共有することを目的として作られたメモ帳アプリです。

## 開発への参加方法

このアプリはXamarinを使って作られています。

そのため、以下のものが必要です。

- mac
- XCode
- Visual Studio for Mac
- iPhone、iPadまたはAndroid実機

なお、Windows上での開発までは検証できていません。
